## 🎨 Şablonlar

### 🥖 Anadolu Toptan
> Mahalle fırını sıcaklığını koruyan, bayi ve sabah sevkiyatı odaklı klasik model.

- Sıcak renk paleti (toprak tonları, turuncu vurgu)
- **Fraunces** serif başlık + **Manrope** gövde tipografi
- 6 kart ürün grid'i, 4 adım süreç, müşteri yorumları, FAQ accordion
- Timeline tabanlı teslimat akışı (03:00 → 07:30)
- Sticky header + canlı "pulse" durum rozeti

### 🏢 Fırın Tedarik
> Okul, otel, zincir kafe ve kurumsal yemekhanelere yönelik B2B tedarik modeli.

- Kurumsal renk paleti (koyu lacivert, turkuaz vurgu)
- **Archivo** heavy başlık + **Inter** gövde tipografi
- 6 kategori kataloğu, 4 adım süreç, 6 kartlı kalite bölümü
- Canlı metrik kartları, case study grid, sertifika rozetleri
- Koyu gradient CTA bölümü + 2 kolon iletişim formu

## ✨ Ortak özellikler

- ✅ **Tam responsive** — 1024px ve 720px breakpoint'leri
- ✅ **No build step** — doğrudan tarayıcıda açılır, dependency yok
- ✅ **Semantic HTML5** — nav, header, footer, section, article
- ✅ **Inline SVG ikonlar** — ek font/asset gerektirmez
- ✅ **Self-contained** — logo ve ana görsel lokalde, ürün görselleri Unsplash CDN
- ✅ **Smooth scroll + hover animasyonları**
- ✅ **Sticky header + backdrop blur**
- ✅ **Ana sayfa navigasyonu** + `|` seperator ile bütüncül menü yapısı
- ✅ **KVKK uyumlu placeholder formlar**

## 🚀 Kullanım

```bash
# 1. Repoyu klonla
git clone https://github.com/altariann/atosweb-template-pack-1.git
cd atosweb-template-pack-1

# 2. Tarayıcıda aç
# Windows
start index.html
# macOS
open index.html
# Linux
xdg-open index.html
```

Yerel bir sunucu istersen:

```bash
# Python 3
python -m http.server 8080
# Node (npx)
npx serve .
```

## 🛠️ Stack

- Saf HTML5 + CSS3 (CSS custom properties, grid, flexbox)
- Google Fonts: Manrope, Fraunces, Inter, Archivo
- Görsel kaynakları: Unsplash CDN + lokal logo
- Avatar: pravatar.cc
- Build/transpile: **yok**
