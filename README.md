# Atosweb Template Pack 1

This repository contains the `Anadolu Toptan` and `Fırın Tedarik` demo templates from the Atosweb HTML template set.

Included paths:
- `index.html`
- `assets/overview.css`
- `anadolu-toptan/`
- `firin-tedarik/`

Note: Image references in this pack use external Wikimedia Commons URLs so the templates open without bundling local image binaries.
