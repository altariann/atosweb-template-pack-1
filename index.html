YAKLAŞIM

Neden bu koleksiyon?

Gerçekçi yapı

Kart görünümünden çıkıp sticky header, dolu footer, hero, ürün grid ve müşteri yorumları dahil gerçek müşteri sitesi kurgusu.

Yerel dil

Metinler ajans jargonundan uzak; bayi, büfe, kantin, otel ve kafe kullanımına yakın, ikna edici ticari Türkçe ile yazıldı.

Net odaklar

Her şablon farklı bir satış yaklaşımını gösterir: sıcak tonlu geleneksel ya da kurumsal B2B. Kopyalamadan alternatif sunar.

Hızlı yayın

Build adımı yok. HTML + CSS dosyasını düzenle, FTP ya da GitHub Pages'e at. Marka içeriğini 30 dakikada doldurursun.

TEKNOLOJİ

Basit, hızlı, bakım kolay

HTML5 + CSS3

Saf, semantic HTML ve modern CSS (grid, flex, custom properties). Framework yok.

Google Fonts

Manrope, Fraunces, Inter, Archivo. CDN üzerinden preconnect ile yüklenir.

Inline SVG

Tüm ikonlar inline SVG, ek font veya asset yüklenmez. Renkler CSS'ten kontrol edilir.

No build

Dependency yok, transpile yok, bundler yok. Dosyayı aç ve çalıştır.

Responsive

1024px ve 720px breakpoint'leri ile mobile-first grid düzeni.

Self-contained

Logo ve ana görseller lokalde, ürün görselleri Unsplash CDN üzerinden.
